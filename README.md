# Helpdesk-Lab (Windows Server 2025)

##  Overview
This project shows how to implement a ticketing system (osTicket) in a WIMP Stack setting. With an emphasis on identity management, role-based access control, and incident triage, it mimics an actual Network Operations Center (NOC) configuration.

##  Tech Stack
- **OS:** Windows Server 2025 (Standard)
- **Web Server:** IIS 10.0
- **Database:** MySQL Server 8.0
- **Language:** PHP 8.2
- **Identity:** Active Directory Domain Services
